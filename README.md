## Current Project

### Observatório da Educação: RS e Santa Maria

A personal end-to-end data project built with public Brazilian education data.

It combines BigQuery, Parquet, dbt, DuckDB, data quality tests, visualization and deployment. The project also led to the identification of inconsistencies in a public data pipeline and an open-source contribution to Base dos Dados.
